Règles de confidentialité

LindApp est edité par Agnesoft,une jeune entreprise congolaise oeuvrant dans le développement des produits informatiques.
Le terme Application s’applique à l’application pour mobile ou tablette Android et IOS nommée « LindApp ».

Données personnelles

Depuis notre Application, nous ne collectons aucune donnée personnelle (par exemple: les noms, les adresses, les numéros de téléphone, les adresses de courrier électronique ou les adresses IP complètes).

Permissions demandées par l’Application

Utilisation des contacts

Nous accedons uniquement aux images pour constituer soit le profile de nos utilisateurs soit les images des articles soit le logo du point de vente soit les images du circulaire.
Nous ne collectons aucune autre information.
Les images sont transmis uniquement à la base de données de l'actuel utilisateur.
