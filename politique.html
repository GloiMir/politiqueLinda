Données personnelles

Depuis notre Application, nous ne collectons aucune donnée personnelle (par exemple: les noms, les adresses, les numéros de téléphone, les adresses de courrier électronique ou les adresses IP complètes).

Permissions demandées par l’Application

Utilisation des images, des contacts et de la position

Nous accedons aux images et au caméra pour constituer soit le profile de nos utilisateurs soit les images des articles soit le logo du point de vente soit les preuves de paiement.
Nous accedons également aux contacts pour créer rapidement plusieurs clients d'un point vente à partir du repetrtoire téléphonique de l'utilisateur
Enfin, nous accedons à la position des utilisateurs lors de la création de point de vente seulement pour stocker le pays et la ville du point de vente et lui proposer les contenus bien filtrés pour notre depôt d'articles, pour les annonces, etc.
Pour la version web, nous accédons à cette position via l'adresse IP du navigateur l'utilisateur étant donné la limite du module (expo-location) utilisé dans la version mobile
Nous ne collectons aucune autre information.
Les images, les contacts et la position sont transmis directement à la base de données de LindApp et ne sont jamais partagés avec qui que ce soit.

Mots de passe et code admin

Les mots de passe et codes admin de tous nos utilisateurs sont cryptés avant d'être jalousement gardés dans la base de données de LindApp.
