Breve explication de LindApp

LindApp est une application mobile innovante et efficace spécialisée dans la gestion des points de vente
LindApp vous permet de créer gratuitement votre point de vente et de suivre de près ses activités grâce aux meilleures technologies du moment
Vous pouvez ainsi gérer à partir des vos articles jusqu'à vos dettes an passant par vos ventes et vos depenses, tout ça en tant réel et dans des fractions de sécondes
LindApp vous genere aussi les documents financiers et les factures de votre point de vente en format pdf partageable, imprimable et télechargable
LindApp vous offre aussi une ouverture au monde en vous permettant la publication et la discussion de vos produits fares dans la partie Circulaires et en vous faisant parvenir aux annonces des entreprises pouvant vous être benefiques
LindApp est sécurisé et très facile à utiliser, pas besoin de compétences techniques ni de matériel spécifique, juste votre SMART phone et votre petite connexion internet
Vous pouvez aussi faire la gestion de votre propre compte et de votre point de vente si vous en êtes administrateur (en posseder le code admin)
Quant au payement, LindApp facture 1 $ par point de vente par mois pour son service dans sa version actuelle et la restriction à ceux qui ne payent pas c'est le non accès aux rapports,factures et notes de crédit en format PDF

De la création de point de vente

Voici la liste complete d'elements que LindApp demande lors de la création d'un point de vente:

  1. Le logo (facultatif) : C'est l'image qui apparait comme logo sur les factures, les notes de credit, la page d'accueil de la partie Caisse et Gestion, les rapports, les fiches de stock, ect
  2. La catégorie (obligaoire) : C'est le genre du point de vente que vous créez, leur liste est exhaustive pour chaque version, LindApp filtre le depôt d'articles d'un point de vente selon sa catégorie
  3. Le nom (obligatoire) : C'est le nom que apparait en tête de la page d'accueil, sur les factures, sur les notes de credit, sur les raaports, sur les fiches de stock, etc
  4. Le code admin (obligatoire) : C'est juste un code du choix de l'utilisateur qui lui permettra dans la suite d'avoir beaucoup d'autorisation que les autres membres du point de vente, il ne doit pas être oublié parceque LindApp n'a pas encore integré la recuperation des codes admin dans sa version acutelle
  5. Le taux d'échange et les 2 devises (obligaoire) : C'est grâce à ces elements que LindApp gerere des rapports bien calculé, la 1iere devise choisie est la devise principale et le 2ieme la devise sécondaire
  6. Le phone admin (obligatoire) : Le numéro de télephone avec le code du pays grâce auquel LindApp contacte le point en cas de besoin
  7. Le mail admin (facultatif) : C'est à ce adresse email que LindApp envoit un message en cas de confirmation de payement, de mauvais payement, et d'autres nouvelles
De tous ces elements, seules les devises du taux d'échange ne changent jamais dans le futur du point de vente
Une fois le point de vente créé, LindApp envoit la clé du point de vente à son créateur et desormais le créateur est consideré garant de tous ceux qui s'inscriront et se connecteront via la clé lui envoyée
La clé doit être gardé jalousement parceque LindApp n'a pas encore integré la recuperation des clé de point de vente sa version actuelle
LindApp considere que le créateur du point de vente a l'âge lui autorisant de posseder un point de vente dans son pays
LindApp crypte le code admin du point de vente avant de le stoquer dans la base de données

De la création de compte utilisateur

Voici la liste complete d'elements que LindApp demande lors de la création d'un compte utilisateur:

  1. L'image de profile (obligatoire) : C'est l'image qui apparait sur les circulaires et notifications de messaages dont l'utilisateur actuel est l'auteur, et aussi sur la tête de la page d'accueil
  2. Le nom (obligatoire) : C'est le nom que apparait comme auteur aux ventes, depenses, dettes, circulaires, messaages, etc dont l'actuel utilisateur est auteur
  3. Le mot de passe (obligatoire) : C'est juste un mot de passe du choix de l'utilisateur qui lui permettra dans la suite de se connecter dans n'importe quel télephone contenant l'application LindApp, il ne doit pas être oublié parceque LindApp n'a pas encore integré la recuperation des mots de passe dans sa version acutelle
  4. La clé du point de vente (obligaoire) : C'est la clé du point de vente auquel l'utilisateur veut s'inscrire
Dans LindApp, le nom d'utilisateur est unique par point de vente
LindApp considere que le créateur du compte utilisateur a l'âge lui autorisant de travailler dans un point de vente dans son pays et qu'il a trouvé légalement la clé du point de vente auquel il s'inscrit
LindApp crypte le mot de passe de l'utilisateur avant de le stoquer dans la base de données

De la connexion de l'utilisateur

Voici la liste complete d'elements que LindApp demande lors de la connexion d'un utilisateur:

  1. Le nom (obligatoire) : C'est le nom sous lequel l'utilisateur s'etait inscrit
  2. Le mot de passe (obligatoire) : C'est le mot de passe avec lequel l'utilisateur s'etait inscrit
  3. La clé du point de vente (obligaoire) : C'est la clé du point de vente auquel l'utilisateur veut se connecter

De la gestiion des articles d'un point de vente

Tous les membres d'un point de vente ont le même pouvoir aux articles de ce dernier
chaque membre d'un point de vente, peut y ajouter un ou plusieurs articles, y voir les localisation des articles, y vendre les articles, y ravitailler les articles, y modifier les articles, y dupliquer les articles et y supprimer les articles
Chaque article doit posseder obligatoirement une image, un nom, une quantité, une unité, un prix de vente unitaire, une dévise de vente, un emplacement dans le point de vente et facultativement une date d'expiration
Après création d'un article, on peut jamais en changer la dévise et l'unité
Les articles ajoutés depuis le depôt d'articles, n'ont pas de date d'expiration et d'emplacement et ont pour quantité 10 unité
Sur la liste affiché, les articles se suivent par ordre de date d'expiration les date d'expiration des articles expirés s'affichant en couleur rouge
Les actions effectuées sur les articles sont repercutées en temps réel à tous les membres d'un même point de vente
